-------------------------------------------------------------------------------------
                                      Описание
-------------------------------------------------------------------------------------
1)Чтобы расчитать значения заданий а) и б), нужно корректно вести данные в поле
"вклад", т.е. значение должно быть числовым и > 0;

2)Выбрать в выпадающем списке значения:a) или б) (для подробной информации об этих
значениях наведите курсор на сам выпадающий список);

3)Ввести в поле 2 числовое значение > 0;

4)Нажать на кнопку "Рассчитать".
